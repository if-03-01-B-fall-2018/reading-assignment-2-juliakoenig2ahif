selector and a declaration block

he selector points to the HTML element you want to style

The declaration block contains one or more declarations separated by semicolons

Each declaration includes a CSS property name and a value, separted ba a colon

Use combinators properly: direct child, descendant, adjacent sibling, sibling

escendant selector (space)
div p {
background-color: yellow;
}

child selector (>)
div > p {
background-color: yellow;
}

adjacent sibling selector (+)
div + p {
background-color: yellow;
}

general sibling selector (~)
div ~ p {
background-color: yellow;
}
